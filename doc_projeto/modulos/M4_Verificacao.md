# Módulo 4: Verificação

## Visão Geral
Oferece um protocolo sistemático para avaliação de fontes e notícias, capacitando o usuário a realizar checagens básicas.

## Protocolo de Verificação (Passamani, 2021)
O módulo ensina técnicas alinhadas a critérios acadêmicos de classificação de fontes:

1.  **Análise de URLs**: Identificação de domínios suspeitos ou imitados.
2.  **Busca Reversa de Imagens**: Como verificar a origem de uma foto.
3.  **Histórico de Versões**: Utilização do Arquivo.pt para ver alterações em páginas ao longo do tempo.

## Avaliação
- Ao final da seção, é apresentado um breve **Quiz** para testar os conhecimentos adquiridos.
