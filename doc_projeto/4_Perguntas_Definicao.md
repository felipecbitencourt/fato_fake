# 4. Perguntas para Definição do Escopo

Para avançarmos com precisão, preciso que confirme ou detalhe os seguintes pontos:

## Conteúdo
1.  **Quais serão os "Módulos" (Abas)?** (Ex: Introdução, Tipos de Fake News, Verificação, Quiz).
2.  **Quantidade de Telas/Interações:** Tem uma estimativa de volume de conteúdo?
3.  **Tipos de Interativo:** Além de perguntas de múltipla escolha, você imagina algum "game" específico? (Ex: Classificar notícias arrastando para "Lixo" ou "Compartilhar").

## Design
4.  **Identidade Visual existente?** Existe um guia de cores ou logo do "Fato ou Fake"?
5.  **Comportamento das Abas:** O usuário pode navegar livremente entre as abas ou é sequencial (bloqueado até terminar a anterior)?

## Técnico
6.  **Pontuação:** O curso exige nota mínima para aprovação no LMS (ex: 70%)?
7.  **Tracking:** Precisamos rastrear o tempo em cada aba ou apenas a conclusão?
